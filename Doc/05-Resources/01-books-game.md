# 📚 Sách Kiến trúc Game / Game Architecture Books

Kiến trúc game là nền tảng kỹ thuật quyết định khả năng mở rộng, bảo trì và hiệu suất của một trò chơi. Dưới đây là các cuốn sách chuẩn mực nhất trong ngành phát triển game.

Game architecture is the engineering foundation that determines a game's scalability, maintainability, and runtime performance. Below are the industry-standard textbooks for game developers.

---

### 1. Game Engine Architecture (Jason Gregory, 3rd Ed.)
*   **Mô tả / Description:** 
    *   *VI:* Cuốn sách chuẩn mực nhất trong ngành. Tác giả Jason Gregory đúc kết gần hai thập kỷ kinh nghiệm tại Midway, Electronic Arts và Naughty Dog để trình bày toàn diện cả lý thuyết lẫn thực hành về cấu trúc của một game engine thương mại hiện đại: game loop, rendering, animation, physics, audio, và asset pipeline.
    *   *EN:* The definitive textbook on game engines. Written by Jason Gregory, drawing from his two decades of experience at Midway, EA, and Naughty Dog. It covers both theory and practice of modern commercial game engine architectures: game loop, rendering, animation, physics, audio, and asset pipeline.

### 2. Game Programming Patterns (Robert Nystrom)
*   **Mô tả / Description:**
    *   *VI:* Cuốn sách tuyệt vời dành cho lập trình viên muốn hiểu cách tổ chức code game khoa học, tối ưu. Cuốn sách làm sáng tỏ các mẫu thiết kế (design patterns) áp dụng trực tiếp trong game như: Command (điều khiển), Observer (sự kiện), State Machine (trạng thái), Component, Spatial Partition, Game Loop, Update Method...
    *   *EN:* An exceptional book for developers looking to write clean, organized, and optimized game code. It demystifies classic and game-specific design patterns: Command, Observer, State Machine, Component, Spatial Partition, Game Loop, Update Method, and more.
*   **Hình thức đọc / How to read:**
    *   *VI:* Tích hợp đọc trực tiếp từng chương ngay trên ứng dụng (đồng bộ dữ liệu từ gameprogrammingpatterns.com).
    *   *EN:* Read chapter-by-chapter directly within this app (synced via proxy from gameprogrammingpatterns.com).

### 3. Multiplayer Game Programming: Architecting Networked Games (Josh Glazer & Sanjay Madhav)
*   **Mô tả / Description:**
    *   *VI:* Hướng dẫn thực tế, giàu mã nguồn về phát triển game nhiều người chơi (multiplayer). Sách dẫn dắt qua các khía cạnh truyền nhận gói tin, đồng bộ trạng thái, giải quyết xung đột mạng, kèm theo ví dụ hai game hoàn chỉnh: một game hành động 2D và một game chiến thuật thời gian thực (RTS).
    *   *EN:* A practical, code-rich guide to architecting multiplayer networked games. Covers packet serialization, state replication, lag resolution, and guides you through two complete game implementations: a 2D action game and a real-time strategy (RTS) game.

    *   *EN:* A comprehensive work deep-diving into the backend architecture of MMO games. Covers authoritative server models, client-side prediction, low-latency state synchronization, lag compensation, and asynchronous database transfers.
