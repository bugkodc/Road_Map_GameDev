## 💻 Kênh YouTube về System Design / System Design Channels

Lộ trình học qua video được khuyên dùng: bắt đầu với **freeCodeCamp** hoặc **ByteByteGo** để nắm nền tảng trực quan, sau đó học **Gaurav Sen** để xây dựng tư duy phản xạ (intuition), rồi học **Hussein Nasser** hay **Arpit Bhayani** để đi sâu vào hạ tầng (infrastructure).

Recommended video learning path: Start with **freeCodeCamp** or **ByteByteGo** for visual foundations, progress to **Gaurav Sen** for design intuition, and study **Hussein Nasser** or **Arpit Bhayani** to master infrastructure internals.

| Kênh / Channel | Cấp độ / Level | Đặc điểm / Key Features |
|:---|:---|:---|
| **ByteByteGo (Alex Xu)** | 🟢 Beginner → Mid | *VI:* Sơ đồ hoạt họa sinh động, giải thích hệ thống lớn (YouTube, Twitter) cực kỳ trực quan và dễ tiếp cận.<br>*EN:* Interactive animated diagrams explaining massive systems (YouTube, Twitter) in a highly visual way. |
| **Gaurav Sen** | 🟢 Beginner → Mid | *VI:* Rất tập trung vào tư duy logic (intuition), giải thích cách phân tích vấn đề trước khi đi sâu vào code và thuật toán.<br>*EN:* Focuses on logic and intuition, explaining how to approach a system problem before writing code or algorithms. |
| **Hussein Nasser** | 🟡 Mid → Senior | *VI:* Đi sâu vào các chi tiết hạ tầng: database internals, proxy (NGINX, HAProxy), HTTP/2 & HTTP/3, connection pooling, network protocols.<br>*EN:* Deep dive into infrastructure internals: databases, proxies, HTTP/2 & HTTP/3 tradeoffs, connection pooling, and network protocols. |
| **CodeOpinion (Derek Comartin)** | 🟡 Mid → Senior | *VI:* Cách tiếp cận thực tế về trade-offs trong thiết kế Microservices, messaging patterns (Kafka, RabbitMQ) và Event-Driven Architecture.<br>*EN:* Practical approach to tradeoffs in Microservices, messaging patterns (Kafka, RabbitMQ), and Event-Driven Architecture. |
| **Arpit Bhayani** | 🟡 Mid → Senior | *VI:* Đi sâu vào cách tự xây dựng các thành phần lõi (caching, databases) và giải bài toán kỹ thuật chi tiết, chính xác.<br>*EN:* Deep dives into building core system components (caching, databases) from scratch with technical precision. |
| **GOTO Conferences** | 🔴 Senior | *VI:* Các bài phát biểu chất lượng cao miễn phí từ các tác giả sách kiến trúc kinh điển và các diễn giả hàng đầu tại hội nghị.<br>*EN:* High-quality free conference talks from classic architecture authors and leading software industry speakers. |
| **InfoQ** | 🔴 Senior | *VI:* Nơi các kỹ sư từ Netflix, Uber, Slack chia sẻ cách giải quyết các sự cố vận hành hạ tầng thực tế trong production.<br>*EN:* Talks from practitioners at Netflix, Uber, Slack, sharing real production incidents and scalability solutions. |

## 📚 Khóa học chuyên sâu / Recommended Premium Courses

### 1. Software Architecture & Design of Modern Large Scale Systems (Udemy)
*   **Giảng viên / Instructor:** Michael (Software Architect)
*   **Nội dung / Focus:** 
    *   *VI:* Hướng dẫn cách phân tích các yêu cầu phi chức năng (non-functional requirements), chọn lựa architectural patterns, và vẽ sơ đồ hệ thống lớn một cách bài bản.
    *   *EN:* Step-by-step guidance on analyzing non-functional requirements, choosing architectural patterns, and diagramming large-scale systems.
*   **Tìm kiếm / Search:** [Udemy Course Page](https://www.udemy.com)

### 2. The Complete Cloud Computing Software Architecture Patterns (O'Reilly)
*   **Nội dung / Focus:**
    *   *VI:* Đi sâu vào các mẫu thiết kế đám mây (cloud design patterns) như Circuit Breaker, CQRS, Event Sourcing, Auto-scaling và cơ chế hoạt động thực tế của chúng trên Cloud.
    *   *EN:* Focuses on cloud design patterns (Circuit Breaker, CQRS, Event Sourcing, Auto-scaling) and how to implement them on modern cloud platforms.
*   **Tìm kiếm / Search:** [O'Reilly Learning](https://www.oreilly.com)
