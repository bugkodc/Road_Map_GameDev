# 🌐 Learning Websites & Platforms

Below is a curated list of online learning resources, knowledge repositories, and system design catalogs from industry experts.

---

### 1. ByteByteGo (Alex Xu)
*   **Description:** Explains complex distributed systems using simple language and vivid, animated system design diagrams. Created by the author of the best-selling System Design Interview book series.
*   **Link:** [bytebytego.com](https://bytebytego.com)

---

### 2. High Scalability
*   **Description:** A repository of real-world architecture case studies from tech giants (Google, Netflix, YouTube...). It deep-dives into how systems scale to support millions of concurrent users.
*   **Link:** [highscalability.com](http://highscalability.com)

---

### 3. Microservices.io (Chris Richardson)
*   **Description:** The most comprehensive microservices patterns catalog. Details core patterns such as Saga, CQRS, Event Sourcing, API Gateway, and Database-per-service with pros and cons.
*   **Link:** [microservices.io](https://microservices.io)

---

### 4. Martin Fowler's Blog
*   **Description:** Personal blog of Martin Fowler — one of the world's most prominent software architects. Contains deep essays on Microservices, Event-Driven Systems, Refactoring, and Domain-Driven Design.
*   **Link:** [martinfowler.com](https://martinfowler.com)

---

### 5. InfoQ
*   **Description:** A premier technology platform presenting in-depth talks and articles written by practitioners at FAANG companies and top startups.
*   **Link:** [infoq.com](https://www.infoq.com)

---

### 6. Cloud Reference Architectures (AWS & Azure)
*   **Description:** Standardized reference architecture libraries from the two largest cloud providers, guiding you on how to implement highly scalable and fault-tolerant cloud systems.
*   **Links:**
    *   [AWS Architecture Center](https://aws.amazon.com/architecture)
    *   [Azure Architecture Center](https://learn.microsoft.com/azure/architecture)

---

### 7. Developer to Architect (Mark Richards)
*   **Description:** A non-profit site offering articles and video lectures (Software Architecture Monday) designed to guide developers transitioning into software architects.
*   **Link:** [developertoarchitect.com](https://www.developertoarchitect.com)
